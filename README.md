# Discimen — AI Risk Assessment & Audit Services

**Independent third-party AI risk evaluation for insurance carriers, MGAs, and enterprises deploying generative and agentic AI.**

*discimen.com* — Latin origin: peril, discriminating difference.

## Overview

Discimen provides the evaluation and audit services that the insurance industry needs to confidently underwrite AI liability coverage. We support the full AI insurance lifecycle: pre-underwriting risk assessment, ongoing compliance monitoring, and claims & loss evaluation.

### Who We Serve

- **Insurance Carriers & Reinsurers** — Independent AI risk evaluation to support underwriting decisions
- **MGAs & MGUs** — Outsourced assessment capability for specialty AI insurance products
- **Enterprises** — AI risk quantification to strengthen governance and demonstrate insurability

### Service Lines

1. **Pre-Underwriting AI Risk Assessment** — Model inventory, architecture review, vulnerability profiling, bias evaluation, compliance mapping, and risk scoring
2. **Ongoing Compliance Monitoring** — Annual pressure testing, drift monitoring, policy-to-practice audits, and regulatory change assessment
3. **Claims & Loss Evaluation** — Root cause analysis, AI vs. human error attribution, claim magnitude validation, and expert witness support

### Industry Focus

Healthcare · Manufacturing · Energy · Automotive · Financial Services

## Technology Stack

- **Framework**: React 18 with TypeScript 5.8+
- **Build Tool**: Vite 5.4
- **Styling**: Tailwind CSS 3.4+ with navy (#1a2332) and gold (#d4af37) theme
- **Routing**: React Router DOM 6.30+
- **Icons**: Lucide React
- **Deployment**: Vercel (discimen.com)

## Quick Start

```bash
npm install
npm run dev     # Development server at localhost:8080
npm run build   # Production build
npm run preview # Preview production build
```
